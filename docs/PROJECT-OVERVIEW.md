# FSM SaaS - Project Overview

## Goal

Build a small–medium SaaS for field service businesses
(lawn care, cleaning, HVAC) with:

- Job / work order management
- Scheduling & dispatch
- Customers & service locations
- Technician view (later)
- Basic reporting

## Target users

- Small to mid-size service businesses (5–30 technicians)
- Owner/dispatcher in office + techs in field

## Non-goals (MVP)

- No full accounting / QuickBooks integration yet
- No multi-tenant billing/Stripe yet
- No mobile app; mobile web only

# Project Roadmap

## Milestone 0 – Planning & Project Setup

- Backend from template, first models, seed data, basic endpoints.

## Milestone 1 – Backend Foundations (FSM)

- Customers, Locations, Jobs, Technicians UI + API.

## Milestone 2 – Frontend Foundations

- Calendar view, drag-and-drop assignments.

## Milestone 3 – Core FSM Workflows

- Map of today’s jobs, technician view basics.

## Milestone 4 – Dispatch, Map & Reporting

- KPIs, error handling, a11y, performance.
